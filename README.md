# OneGear Project

## Introduction
OneGear is an e-commerce website project dedicated to Gaming Gear (keyboards, mice, headphones). The website is built to provide a complete online shopping experience with a user-friendly and modern interface.

## Key Features
The project includes all essential functional pages for an e-commerce site:
- Home: Displays banners and featured products.
- Products: Detailed product listings.
- Cart & Checkout: Cart management and order processing.
- Account: Login, registration, and order history viewing.
- News and Contact pages.

## Technologies Used
The project utilizes foundational Front-end technologies:
- HTML5: For website structure.
- CSS3: For interface design and styling.
- JavaScript: For handling user interaction logic.

## Installation
1. Clone the repository to your local machine:
   git clone https://github.com/trthtuan/OneGear-Project.git
2. Navigate to the project directory.
3. Open the index.html file in your browser to launch the website.
